This application allows for the creation and managment of various lits of chores. These chores may then be performed by the the children of the parent user, who can create the accounts for them, to redeem for points. These children users may then manage these points and redeem them for various rewards.

This application will, at this time, be hosted on the UMW public webserver, and make use of several internal databases to manage and track these acounts and lists. It is a non-profit application that uses a combination of public access code/applications and original code, done mostly in python.

Code created by: Michael Bloeker, James Bramnick, Shawn McElroy

Stored on GitHub and Cloud9

Written in: python, html, css, javascript

Open source code: W3B school, *Google API*
